## Suggested pace

- **Full-time:** 8–12 weeks (all 16 modules)
- **Part-time:** 4–6 months
- **Reference:** jump to any module; use [Topic Map](topic-map.md) for concepts and [Resources](resources/index.md) for papers and tools
