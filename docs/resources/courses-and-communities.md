# Courses & Communities

Free and paid resources to supplement the handbook.

## Free courses

| Resource | Format | Best for |
|----------|--------|----------|
| [Microsoft AI Agents for Beginners](https://github.com/microsoft/ai-agents-for-beginners) | GitHub + videos | Agent fundamentals |
| [Fast.ai](https://www.fast.ai/) | Notebooks | Deep learning intuition |
| [Hugging Face Course](https://huggingface.co/learn) | Interactive | Transformers, fine-tuning |
| [DeepLearning.AI short courses](https://www.deeplearning.ai/short-courses/) | Video + labs | RAG, agents, LangChain |
| [Full Stack LLM Bootcamp](https://fullstackdeeplearning.com/llm-bootcamp/) | Videos | Production LLM apps |
