# Maintaining the handbook

This site is organized for **learners** (simple sequential nav) and **contributors** (one config file + scripts).

## Architecture

```
curriculum.yml          ← single source of truth for nav order & display titles
scripts/generate-nav.mjs ← builds mkdocs.yml nav + learn/index.md + learning-path.md
docs/{phase}/module-NN-{slug}/   ← on-disk folders (synced from platform; IDs hidden in UI)
```

**Top nav tabs:** Home · Start Here · Learn · Reference · Projects · Contribute

**Learn tab:** 16 numbered courses (index pages only). Lessons are listed on each course page — not in the sidebar.

## Add a new course

1. Add content under the right phase folder, e.g. `docs/build/module-20-my-topic/index.md` + `lessons/`.
2. Edit `curriculum.yml` — insert a block under `courses` in the order learners should take it:

```yaml
  - title: My New Topic
    path: build/module-20-my-topic
```

3. Regenerate navigation:

```bash
npm run sync-nav
```

4. Open the course index page and ensure the lesson table lists all lessons.
5. Run `mkdocs serve` and click through the new entry in **Learn**.
6. Update [Topic Map](docs/topic-map.md) or [Start Here](docs/start-here.md) if the topic is a major entry point.

## Add an optional track

Tracks are short focused paths (like Agent Engineering). Add under `tracks` in `curriculum.yml`:

```yaml
tracks:
  - title: My Track
    path: my-track-folder
    pages:
      - { title: Overview, file: index.md }
      - { title: Lesson One, file: 01-lesson.md }
```

Then `npm run sync-nav`.

## Sync from AI Engineering platform

If content is migrated from the sibling platform repo:

```bash
npm run migrate
```

This copies lessons and calls `syncNav()` — **do not** hand-edit `mkdocs.yml` nav; it will be overwritten.

## Files you should edit

| File | When |
|------|------|
| `curriculum.yml` | New course, reorder path, rename display title |
| `docs/**/lessons/*.md` | Lesson content |
| `docs/learn/study-plans.md` | Week-by-week schedules (manual) |
| `docs/start-here.md`, `docs/topic-map.md` | Routing / concept lookup |

## Files generated — do not edit by hand

| File | Generated by |
|------|----------------|
| `mkdocs.yml` (nav section) | `npm run sync-nav` |
| `docs/learn/index.md` | `npm run sync-nav` |
| `docs/learning-path.md` | `npm run sync-nav` |
