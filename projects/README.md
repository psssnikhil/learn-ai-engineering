# Capstone Projects

Portfolio-worthy builds from the Advanced phase. Full specs live in [module 17](../docs/advanced/17-capstone-projects/index.md).

## Projects

1. **Enterprise RAG System** — document intelligence, hybrid search, citations, API
2. **AI Code Review Agent** — multi-agent PR analysis with GitHub integration
3. **Research Assistant** — MCP-powered paper search and report drafting
4. **Customer Support Bot** — RAG + intent routing + human handoff
5. **Content Pipeline** — multi-agent research, draft, edit, fact-check
6. **Personal AI Assistant** — memory, tools, persistent context

## How to use

- Pick **one** project aligned with your goals
- Complete relevant Build-phase modules first (RAG, Agents)
- Ship something deployable — README, tests, and a demo matter more than feature count

Details and tech stacks are in each capstone lesson in module 17.
